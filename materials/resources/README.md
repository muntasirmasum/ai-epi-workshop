# Additional Resources

Guides, references, and learning materials to support your AI-assisted statistical analysis journey.

## Quick Start

See the [Quick Start Guide](../quick-start-guide.md) for setup instructions and workshop preparation.

## Learning R

- **R for Data Science (free book):** https://r4ds.hadley.nz/
  - Comprehensive introduction to R and tidyverse
  - Clear examples and exercises
  - Perfect for beginners

- **RStudio Education:** https://education.rstudio.com/
  - Tutorials and learning paths
  - Video lectures
  - Interactive exercises

- **Swirl (learn R in R):** https://swirlstats.com/
  - Interactive lessons inside R
  - Practice as you learn
  - Immediate feedback

- **RStudio Cheatsheets:** https://posit.co/resources/cheatsheets/
  - Quick reference sheets for Base R, dplyr, ggplot2, and more

## Statistics & Epidemiology

- **OpenIntro Statistics:** https://www.openintro.org/book/os/
  - Free introductory textbook
  - Focuses on applied statistics

- **Epidemiologic Methods:** https://www.hsph.harvard.edu/miguel-hernan/causal-inference-book/
  - Hernan & Robins textbook (free online)
  - Causal inference focus

- **Stat 545 (UBC course):** https://stat545.com/
  - Data wrangling and visualization
  - Excellent R tutorials

## AI Tools

- **Claude AI:** https://claude.ai
  - Free tier available
  - No coding required to use
  - Good for Q&A and explanations

- **Claude Desktop:** https://claude.ai/download
  - Desktop application for Mac/Windows
  - Integrates with ClaudeR package

- **ClaudeR Package:** https://github.com/IMNMV/ClaudeR
  - Direct RStudio integration
  - GitHub repository with documentation

## NHANES Data

- **NHANES Website:** https://www.cdc.gov/nchs/nhanes/
  - Access all NHANES datasets
  - Codebooks and documentation
  - Analysis guidelines

- **NHANES R Package:** https://cran.r-project.org/package=NHANES
  - Pre-loaded NHANES data in R
  - Easy access to variables

## Software Alternatives

If you can't install software on your computer:

  - No installation required
  - Free tier available
